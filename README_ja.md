# 二元論の原罪

**言語 / Language:** 日本語 | [English Version](README.md)

## The Original Sin of Dualism

**科学・文明・人間思想は、なぜ未来を失ったのか**

---

## 概要

**二元論の原罪**は、西洋近代文明の基盤に埋め込まれた二元論を、単なる哲学上の誤りではなく、科学、技術、政治、宗教、経済、文化進化の方向性を歪めてきた文明的バイアスとして分析する白書である。

ここでいう「原罪」とは、特定の宗教的断罪ではない。  
それは、人類文明が自然と人間、自己と他者、善と悪、支配と服従、勝者と敗者を分離し、対立構造の中で世界を理解するようになった根本的な認識エラーを指す。

本稿の中心命題は、次である。

> 技術は中立である。  
> しかし、文明の進路は、その根底にある哲学によって決定される。

二元論は、科学を調和の道具ではなく、支配の道具へと変形させた。  
一方で、**和**――調和、循環、共存、自然法則への適合を重視する思想――に基づく文明であれば、科学はまったく別の方向へ進んでいた可能性がある。

したがって、人類の未来は、新しい発明だけによって決まるのではない。  
人類が自らの基盤となる世界観を変換できるかどうかにかかっている。

## 要旨

本稿は、現代文明の危機を、単なるCO₂排出、経済格差、政策不全、技術不足の問題としてではなく、文明の根本哲学の問題として捉える。

西洋近代文明は、長い時間をかけて、世界を二項対立で理解する認識構造を強化してきた。

- 善 / 悪
- 自己 / 他者
- 勝者 / 敗者
- 人間 / 自然
- 文明 / 野生
- 支配 / 服従
- 進歩 / 停滞

この二元論的認識は、科学技術の方向性をも決定した。

核科学は原子爆弾へ、化学は毒ガスや生物兵器、枯葉剤へ、航空技術は爆撃機や戦闘機へ、AIは監視、自律兵器、心理操作へと向かいやすくなった。

これは偶然ではない。  
技術は、それを使う文明の思想に従う。

本稿では、二元論が作り出した破壊的科学の軌道と、それに対する代替原理としての**和**、さらに六つの理――摂理・調和・循環・構造・秩序・和――を提示する。

## 1. 二元論：人類文明を歪めた構造

二元論とは、世界を対立する二つの項に分けて理解する認識様式である。

それは、分析の道具としては一定の有効性を持つ。しかし、それが文明の基盤となると、世界は関係性の場ではなく、勝敗を決める戦場として理解される。

### 二元論的認識の例

- 善と悪
- 正義と敵
- 人間と自然
- 文明と野蛮
- 信者と異教徒
- 支配者と被支配者
- 成長と停滞
- 勝者と敗者

この認識構造は、科学の進路を長い時間にわたって方向づけてきた。

**二元論 → 支配の科学**

科学的発見そのものは中立であっても、文明の思想が支配を前提にしていれば、発見は支配、制御、軍事、競争へ向かう。

---

## 2. 科学が支配の道具になった事例

### 2.1 核科学

核分裂の発見は、自然法則の理解としては中立であった。しかし、最初の大規模応用は原子爆弾だった。

平和利用は、その後に「道徳的修復」として語られるようになった。これは、科学そのものの性質ではなく、文明の優先順位の問題である。

### 2.2 化学

化学の本来の目的は、物質の性質と相互作用を理解することである。

しかし、二元論的文明の中では、毒ガス、生物兵器、枯葉剤、難分解性汚染物質などが優先的に研究・利用されてきた。

自然循環や微生物との調和よりも、制御、効率、軍事的優位が優先された。

### 2.3 航空技術

空を飛ぶという人類の願いは、本来、移動、救助、交流、観測の可能性を広げるものだった。

しかし、航空技術は早い段階で爆撃機、戦闘機、軍用ドローンへと向かった。

飛行は「つなぐ技術」であると同時に、「攻撃する技術」として発展してしまった。

### 2.4 人工知能

AIの本来の可能性は、人間の知性を補完し、環境観測、合意形成、教育、医療、災害対応、文明設計を支援することにある。

しかし、二元論的な競争構造の中では、監視、自律兵器、大規模心理操作、情報支配、労働搾取へ向かいやすい。

AIそのものが悪なのではない。  
AIを支配の文明OSで運用することが問題なのである。

---

## 3. 技術は思想に従う

これらの例は、偶然ではない。

技術は、単独では方向を持たない。  
しかし、文明は技術に方向を与える。

二元論的文明では、科学技術は次のような流れを取りやすい。

1. 自然法則を発見する。
2. それを支配・制御の手段として解釈する。
3. 軍事化する。
4. 商業化する。
5. その後に、平和利用を検討する。

これに対し、和に基づく文明では、発見は最初から共存、循環、回復、生命維持、調和のために使われる。

科学は「何ができるか」だけで定義されるのではない。  
科学は「何のために使うべきか」という文明哲学によって定義される。

## 5. 江戸期日本：非二元的科学の証拠

江戸期日本は、完全な理想社会ではなかった。しかし、非二元的な科学文明の歴史的参照点として重要である。

### 和算

和算は高度な数学でありながら、砲術や弾道学、軍事支配のために中心化されなかった。商業、建築、測量、農業、工学など、生活と構造に応用された。

### 医学

医学は、主に治療と回復を目的とした。生物兵器や非倫理的実験へ向かう思想的圧力は、近代軍事科学ほど強くなかった。

### 天文学

天文学は、暦の精度、農業、季節理解に用いられた。征服や宗教的支配の道具としてよりも、生活の調律として機能した。

### 金属加工

高度な金属技術は、道具、工芸、美、生活用品へ向かった。大量破壊兵器としての展開が中心ではなかった。

これらは、基盤思想が調和であるとき、科学は建設的な力として発展し得ることを示している。

## 9. 六つの理：文明存続の前提条件

人類が進むべき道は、すでに明確である。

1. **摂理 / Natural Law**  
   自然法則へ戻る。

2. **調和 / Harmony**  
   人間、自然、技術、社会、AIの調和を取り戻す。

3. **循環 / Circulation**  
   水、炭素、栄養、有機物、エネルギー、情報の循環を回復する。

4. **構造 / Structure**  
   循環を実装する都市、農業、海洋、エネルギー、物流、AI観測網を再構築する。

5. **秩序 / Order**  
   破壊、暴走、過剰集中、乱獲を防ぐ安定原理を持つ。

6. **和 / Wa**  
   対立ではなく、補完と共存によって統合する。

これらは単なる哲学ではない。  
文明そのものが存続するための前提条件である。
